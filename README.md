# 🎬 CWD Oscar Formula

A Clarity With Data (CWD) project exploring two questions:

1. What helps movies win Oscars?
2. What happens after actors win?

## Project Components

- **The Oscar Formula** — patterns behind Best Picture winners and nominees
- **The Oscar Curse** — pre/post-win career analysis for actors

## Repository Structure

- `data/raw/` - source data pulls and scraped data
- `data/processed/` - cleaned and analysis-ready data
- `notebooks/` - exploratory and analytical notebooks
- `src/` - reusable helper functions
- `app/` - Streamlit interactive app
- `outputs/` - exported charts and tables
